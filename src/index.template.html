  1. 先改善散热：安装并使用上风压散热器。

用于进入 AMD PBS / CBS / Overclocking 隐藏菜单，修改内存频率、时序等真实固件参数；错误设置可能导致无法启动。

本机约 80MB 的 UMAF 启动分区不可删除或格式化。

进入：开机或重启时持续按 F2，选择最右边第三项，再进入后续界面的第二项；详细安装与进入方法见第 03 章教程 01。

04

异常恢复

先处理 UXTU 异常；需要回到本文优化基线或彻底恢复默认时，再按下方对应路径操作。

UXTU 没自启，手动打开也没反应

删除 AppData\Local\JamesCJ60 下截图红圈所示的 Universal x86 Tuning Utility 配置文件夹。删除后重新打开 UXTU，并开启“开机自启”和“开机自启自动应用配置”。

恢复到本文记录

回到最初优化调校

  1. 固件前置：关闭 CCD1、完成 BIOS 解锁、安装并进入 UMAF；参考第 03 章教程 01。
  2. CPU 与功耗：关闭 CPU 睿频、调出高性能电源计划并调整功耗限制；参考第 03 章教程 02，以及第 01 章的调校说明和第 03 章里的现有截图。
  3. 内存：内存频率和时序控制参考附录里的参数。
撤销全部调校

完全恢复默认

  1. 关闭 UXTU 开机自启；再在任务栏图标上右键退出 UXTU，取消当前全核负压。
  2. 参考第 03 章教程 02 里的注册表设置重新开启 CPU 睿频；在电源计划里把定频改为 0。
  3. 机械革命控制台里不再使用自定义功耗模式；游戏时改用静音模式或狂暴模式，非游戏场景使用平衡模式。
  4. 参考第 03 章教程 01 进入 UMAF，把 CCD1 重新打开，恢复 16 核 32 线程。
  5. 在 UMAF 里把内存超频设为 Disabled，恢复 5200 MT/s / C42 的默认内存设置；参考第 03 章教程 03。
